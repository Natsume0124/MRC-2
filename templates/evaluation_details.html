Submissions will be evaluated based on (Preliminary):
  • Accuracy in Visual Question Answering

    We employ the large language model to assist in evaluating the free-text similarity between the model's predictions and the ground-truth answers.

  • Grounding Accuracy

    Evaluating visual grounding using the standard ACC@0.5 protocol, where a prediction is considered correct if its IoU with the ground-truth bounding box exceeds 0.5.
